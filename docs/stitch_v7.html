Booting…
The thesis: scale by sharding domains, not by growing one model. Six independent 114M ternary experts (~57 MB each), distilled from Qwen 2.5 0.5B-class teachers on their own domain. Architecturally each expert can live on a different machine — only the token id and the 132-dim meaning vector cross the wire. Experts load on demand: the classifier picks one per token, and we only fetch a model the first time we need it. Use the latency selector to feel what a real distributed deploy would be like.
